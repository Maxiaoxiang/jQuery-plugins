缩放-resizable

需先引入jQuery，再引入resizable组件

<script src="jquery.js"></script>
<script src="jquery.resizable.js"></script>

组件样式与功能分离，自定义样式

HTML

<div class="test"></div>

Javascript

$('.test').resizable();

HTML

<div class="range">
    <div class="test1"></div>
</div>

Javascript

$('.test1').resizable({
    rangeCls:'range'
});

options(参数配置)

参数 默认值 说明
rangeCls '' 拖动最大范围
startResize function(){} 开始缩放事件
resizeing function(){} 缩放时事件
stopResize function(){} 停止缩放事件

api接口

方法 说明
start 开始缩放
getFlag 返回缩放状态
stop 停止缩放，isReszeing必须为true，将设置为false
getMouseCoords 返回鼠标坐标
