弹出层-dialog

下载该插件

需先引入jQuery，再引入dialog组件

<script src="jquery.js"></script>
<script src="jquery.dialog.js"></script>

组件样式与功能分离，自定义样式

HTML

<div class="M-box"></div>

Javascript

$('.dialog1').click(function(){
    $('.M-box').dialog({
        title:'标题',
        content:'内容',
        clickMask:true,
        isDraggable:true,
        button:{
            '确定|ok':function(api){
                alert('确定前回调');
                api.close();
            },
            '取消|cancel':function(api){
                api.close();
            }
        },
        beforeShow:function(){
            alert('显示前事件');
        }
    });
});

    });
});

$('.dialog4').click(function(){
    $('.M-box').dialog({
        title:'标题',
        content:'内容',
        isDraggable:true,
        clone:true,
        isResizable:true
    });
});

options(参数配置)

参数 默认值 说明
title '' 标题
close 'x' 关闭按钮内容
content '' 内容
zIndexplain 9999 弹窗层级
isMask true 是否开启遮罩
clickMask false 是否可以点击遮罩关闭弹窗
opacity 0.5 遮罩透明度
button {} 按钮配置，例：'确定|ok':function(){}，名字|样式名+回调，多个按钮用逗号隔开，如果没传样式默认样式为'M-btn'
type 'dialog' 值为'message'时无title，button，close并且自动关闭
time 2000 type为message时多少秒后自动关闭
followScroll true 弹窗是否跟随屏幕滚动
isDraggable true 是否能拖动
handleCls 'M-title' 拖动把手的class，默认为标题class
axis '' 拖动方向，值为：x / y
rangeCls '' 范围，默认为视图窗口
clone false 克隆拖动，默认为false
cloneCls 'clone' 克隆拖动元素样式名
isResizable false 缩放
beforeShow function(){} 显示前的事件
afterHide function(){} 关闭后的事件

api接口

方法 说明
open 打开弹窗
close 关闭弹窗
getOpen 获取弹窗是否打开，isOpen
scroll 窗口滚动居中
resize 窗口变化居中
getDraging 返回拖动状态
stop 停止拖动
getReszeing 返回缩放状态
resizeStop 停止缩放
